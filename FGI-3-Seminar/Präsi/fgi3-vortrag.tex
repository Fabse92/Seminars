\section{ Iterierte Umfärbungen }

\begin{frame}
  \frametitle{ Färbungen und Umfärbungen }


  Wir nehmen eine vorgegebene Mengen an Farben $C$ an.

  Der Einfachheit halber identifizieren wir eine Tüte $T$ mit $n$
  Gummibären mit dem Intervall $[1, \ldots, n]$.


  \begin{definition}[Färbung]
    Eine \emph{Färbung} ist eine Abbildung $f: [1, \ldots, n] \to C$.

    Sei $F$  die Menge aller Färbungen
  \end{definition}

  \begin{definition}[Umfärbung]
    Eine \emph{Umfärbung} ist eine Funktional $u: F \to F$.
  \end{definition}


  Anmerkung: Wir wollen \emph{Funktional} nicht formal einführen;
  informal ist es eine Funktion, die Funktionen als Argumente hat
  (Details in der Ausarbeitung).

\end{frame}


\subsection{  Iteration, Stabilisation }

\begin{frame}
  \frametitle{  Iterierte Umfärbungen }

  Sei die Färbung $f: [1, \ldots, n] \to C$ gegeben.

  \begin{definition}    
    Die Iteration einer Umfärbung $u: F \to F$ ist
    \[
    \begin{array}{rcl}
      u^0(f) &:=& f \\
      u^{n+1}(f) &:=&     u(u^{n}(f)) 
    \end{array}
    \]
  \end{definition}

  \pause

  Wir hätten es gerne, dass sich $u^{n}(f)$ für $n \to \infty$
  stabilisiert.


\end{frame}



\begin{frame}
  \frametitle{  Stabilisierung iterierter Umfärbungen }

  Wir hätten es gerne, dass sich $u^{n}(f)$ für $n \to \infty$
  stabilisiert.

  \begin{definition}[Stabilisierung]

    Die Iteration einer Umfärbung $u: F \to F$ stabilisiert sich,
    wenn .....
  \end{definition}


\end{frame}


\begin{frame}
  \frametitle{  Beispiele zur Stabilisierung }

  Manchmal stabilisiert sich $u^{n}(f)$ für $n \to \infty$:


  \begin{example}
    Sei ....
  \end{example}

  \pause

  Das sich $u^{n}(f)$ nicht immer stabilisiert, zeigt folgendes
  Beispiel:
  \begin{example}
    Sei ....
  \end{example}

\end{frame}



\begin{frame}
  \frametitle{  Umfärbungsensembles }
  
  Angenommen wir haben die Menge von Umfärbungen $U = \{u_1, \ldots
  u_k: F \to F\}$, genannt \emph{Umfärbungsensemble}.

  Dann können wir $U$ auch iterieren, indem wir im $n$-ten Schritt
  eine der $k$ Umfärbungen anwenden.


  Formal haben wir dazu eine Abbildung $g: \mathbb{N} \to [1, \ldots,
  k]$, die im $n$-ten Schritt die Umfärbung $u_{g(n)}$ auswählt.

  \begin{definition}
    Die \emph{Iteration von $f$ durch das Umfärbungsensemble $U$ mit
      Auswahl $g$} ist:
    \[
    \begin{array}{rcl}
      U_g^0(f) &:=& f \\[3mm]
      U_g^{n+1}(f) &:=&     u_{g(n+1)}(U_g^{n}(f)) 
    \end{array}
    \]
  \end{definition}


\end{frame}

\subsection{  Eindeutigkeit  }



\begin{frame}
  \frametitle{ Das Problem der Eindeutigkeit }

  Es gibt eine Besonderheit des Wohlordnungssatz auf Gummibärenfarben:


  Der Wohlordnungssatz auf Gummibärenfarben garantiert die Stabiliserung.

  Er garantiert aber nicht die Eindeutigkeit des Endergebnisses.

  Das Endergebnis hängt von der Auswahlfunktion $g: \mathbb{N} \to [1,
  \ldots, k]$ auf dem Umfärbungsensemble ab.

  \begin{block}{Frage}
    Für welche Umfärbungsensembles ist auch das Endergebnis eindeutig?
    
  \end{block}

\end{frame}
